1-6月：

  • 算法精进
    • Hot 100
    • 剑指 offer 算法题

    • HTML / CSS

    • 框架 Vuejs：常见题目
    • 框架 Reactjs：常见题目

    • 设计模式：发布订阅，观察者、装饰器、适配器
    • HTTP 和网络：状态码、HTTP 1.0/1.1/2.0/3.0、握手和挥手、tcp和udp、websocket

      • vuejs 3.0、React.js 18
      • 微前端多种方案；

7-12月：

  • Nodejs 精进
  • Go / Java 实战提升
