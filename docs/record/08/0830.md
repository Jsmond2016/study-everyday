- Typescript 中 高级类型 Exclude 的使用

对比 Pick 和 Omit 的使用：

```typescript

type A = {
  name: string;
  age: number;
}

type C = {
  name: string
}

// 可以使用 Pick 抽取某个属性
type CC = Pick<A, 'name'> // 对应的类型为 const c = { name: string }

// 可以使用 Omit 除去某些属性

type CCC = Omit<A, 'age'> // 对应的类型为 const c = { name: string }

```

但是，如果类型是这种：

```typescript

type A = 1 | 2 | 3

type B = 3

type BB = Extract<A, 3>
type BBB = Exclude<A, 1 | 2 >
```

当然，表示某个属性，也可以这样用：

```typescript
type A = {
  name: string;
  age: number;
}

type B = {
  nickName: A['name']
  realAge: A['age']
}

```
