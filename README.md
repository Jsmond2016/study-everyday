## 创建总结

```bash

# 创建日总结模板
# 爬虫自动抓取 bing 壁纸作为今天的主图

yarn new 

# or

yarn new week # 创建周总结模板

# or 生成指定日期的 日-周 模板

yarn new day 0901

yarn new week 0902
```
