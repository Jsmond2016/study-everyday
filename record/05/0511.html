# 学习

  • 工具使用：nodemon

    • nodemon/github
    • Nodemon 介绍和使用
    • nodemon使用方法

  • 最简单的配置方式：

    • package.json: nodemon ./src/app.js
    • nodemon.json 文件：
    {
  "watch":["./src/**/*.*"]
}

    这样就可以监听 src 目录下所有文件变更并且自动重启
