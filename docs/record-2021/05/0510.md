  - 函数内需要等待异步操作的返回结果时，使用 `promise` ：遇到一个函数A调用函数B，关系大致如下：
  
  ```js
  function A() {
      // 这里的 b 是 undefined
      const b = B()
  }
  async function B() {
      const r1 = await getR1()
      const img = new Image()
      img.onload = function () {
          //...
          const r2 = xxx()
          return r2
      }
  }
  ```
  
  看上面的代码，拿到的结果 b 是空的，因为下面函数 B 返回是 void，虽然异步操作 `onload` 最后有返回，但是因为是异步，因此拿不到。
  
  改进：
  
  ```js
  async function A() {
      // 这里的 b 是 有值的
      const b = await B()
  }
  async function B() {
      return new Promise((resolve) => {
          const r1 = await getR1()
          const img = new Image()
          img.onload = function () {
              //...
              const r2 = xxx()
              resolve(r2)
          }
      })
  }
  ```
  
  不知道为啥自己一时间脑子短路，居然忘了异步返回怎么处理，诶！！
